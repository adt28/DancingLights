DancingLights - Python Arcade
By A.D.Tejpal - 01-Apr-2021

Demonstrates display of animated lights by using the module:
arcade.experimental.lights

There is a soft light object at center of a light wheel, having six normal light objects at its periphery.

The ligh objects pulsate between max / min size, while the light wheel as a whole keeps moving all over the screen, at the same time, rotating around its axis.
